Astronomy Picture of the Day

Project Info:

For this capstone project, I applied my JavaScript skills to create a dynamic single-page application that fetches and displays data from NASA's Astronomy Picture of the Day API. Using vanilla JavaScript, I utilized innerHTML to seamlessly update the content without relying on frameworks.

Initially, I aimed to use Turn.JS for a photo album of favorite images, but integrating dynamic content proved challenging. Instead, I adapted it to function as a gallery, similar to a previous assignment. Some CSS issues still need addressing.

Future enhancements include adding CSS animations such as twinkling stars, a moving telescope, and an interactive light switch. I may also explore using Rive for more advanced animations.

This project taught me the basics of API integration, DOM manipulation, and resolving CSS challenges. It highlighted areas for further learning and improved my ability to communicate design functionality with developers.
