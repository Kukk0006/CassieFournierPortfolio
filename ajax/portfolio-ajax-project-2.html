TrailWise - An Educational Hiking App Design

Project Info:

In a team of five we were tasked with creating an application from start to finish. Our problem is that trail apps were not as engaging and educational. Our process included:

  • Discover the problem
  • Research
  • Personas, User Flows and Journeys
  • Interactive Lo-Fi Wire frames
  • Focus Group with Our Peers
  • User Test using Maze
  • Hi-Fi Prototype
  • Present in a Showcase

Project Details:

  • Course:UX Design II - Level 3
  • Roles:Designer
  • Software/hardware:Photoshop, Figma, FigJam, Maze
